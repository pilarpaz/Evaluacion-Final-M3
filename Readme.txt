Documentos que incluye el proyecto
Carpeta html 
•	index.html / Página de inicio, contiene información de los objetivos del sistema de información.

•	contacto.html / Sitio estático, contiene formulario con los siguientes campos: nombre, correo electrónico,  teléfono, mensaje.

•	crearcapacitacion.html  /  Sitio estático, contiene formulario para una posible capacitación contiene los siguientes campos: fecha de capacitación, hora de capacitación,
lugar, duración, y nombre de cliente.

•	crearusuario.html  /  Sitio estático, contiene formulario para crear usuario con los siguientes datos: nombres, los apellidos, la fecha de nacimiento, el RUN y el tipo de
usuario que se desea registrar

•	editarcliente.html  /  Sitio estático, contiene formulario que permite modificar datos de usuario considera los campos: RUN cliente, nombres y apellidos del cliente, teléfono, AFP, sistema de salud, dirección, comuna y edad

•	editarprofesional.html  /  Sitio estático, contiene formulario que permite modificar los datos del usuario considera los campos: RUN profesional, nombres y apellidos de este, el teléfono, el título profesional y el nombre del proyecto en el que se desempeña, se accede a esta página a través del listado de usuario

•	editaradministrativo.html  /  Sitio estático, contiene formulario que permite modificar los datos del usuario considera los campos: RUN administrativo, nombres y apellidos del administrativo, correo electrónico y nombre de área a la que pertenece, se accede a esta página a través del listado de usuario

•	listadousuarios.html /  Listado con usuarios existentes en la plataforma contiene los siguientes campos: RUN, nombre y apellidos del usuario, e incluye a qué tipo de usuario pertenece (cliente, administrativo o profesional)

•	listarcapacitacion.html  /  Despliega listado de capacitaciones con los siguientes datos: id de capacitación, fecha de capacitación, hora de capacitación y nombre del cliente.

Imágenes responsivas utilizadas en sitio web.

Sitio web desarrollado para empresa de asesoría en prevención de riesgos. Posee un sistema que permite almacenar  y administrar la información ingresada con la finalidad de planificar y coordinar visitas a terreno.
Sitio web responsivo adaptado para ser visualizado tanto en Computadores, Tablet y dispositivos moviles
